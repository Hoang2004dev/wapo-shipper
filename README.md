# Fashion Shipper — Demo

Scaffold Vite + React (JavaScript) for a Shipper UI.

This project expects an existing C# backend. Configure the backend base URL via the environment variable `VITE_API_BASE` (for example `VITE_API_BASE=http://localhost:5000`).

Quick start

Install dependencies:

```bash
npm install
```

Run dev server:

```bash
npm run dev
```

API placeholders are provided in `src/services/api.js`. Replace the endpoints with your C# API routes as needed.
